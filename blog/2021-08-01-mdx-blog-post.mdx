## Examples

### Notes

Messages begin with three colons `:::` + an identifier and end with three colons `:::`, each on its own line.

:::note
Regular note, identifier **note**
:::

:::tip
Hint-type note, identifier **tip**
:::

:::info
Info to think about, identifier **info**
:::

:::warning
Requires attention, identifier **warning**
:::

:::danger
Error, all lost, identifier **danger**
:::

### Quotes

Quotes begin with the '>' character, which begins a line.

> This is a quote (line 1)
> Continue quoting (line 2)

### Details

The `<details>` tag is used for the block, with a `<summary>` tag inside for the block title.
Collapsible text supports MarkDown markup.

<details>
<summary>**Collapsible text title**</summary>

For the title to work, the collapsible text itself must be separated from the title by a space, or
the text following the title is a list.
Draw a line and add a message after it.
***
:::note Warning
This is an **added** message.
:::
</details>
